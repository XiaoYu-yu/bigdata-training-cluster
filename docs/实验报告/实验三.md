# 实验三：Sqoop数据导入/导出

## 实验目的

- 熟悉Sqoop的基本使用方法
- 掌握Hive与MySQL之间的数据导入/导出
- 了解大数据与关系型数据库之间的数据交互

## 实验步骤

### 1. 准备工作

1. **检查Sqoop版本**：`sqoop version`
2. **验证MySQL连接**：`sqoop list-databases --connect jdbc:mysql://localhost:3306 --username root --password 123456`

### 2. 创建Hive内部表

1. **创建内部表**：
   ```sql
   create table dbtaobao.inner_user_log(
       user_id string,
       item_id string,
       cat_id string,
       merchant_id string,
       brand_id string,
       month int,
       day int,
       action int,
       age_range int,
       gender int
   )
   row format delimited fields terminated by ',';
   ```
2. **插入数据**：`hive -e "insert into dbtaobao.inner_user_log select * from dbtaobao.user_log;"`
3. **验证数据行数**：`hive -e "select count(*) from dbtaobao.inner_user_log;"`

### 3. 从Hive导入到MySQL

1. **创建MySQL表**：
   ```sql
   create database if not exists dbtaobao;
   use dbtaobao;
   create table user_log(
       user_id varchar(20),
       item_id varchar(20),
       cat_id varchar(20),
       merchant_id varchar(20),
       brand_id varchar(20),
       month int,
       day int,
       action int,
       age_range int,
       gender int,
       primary key(user_id, item_id)
   ) engine=InnoDB default charset=utf8mb4;
   ```

2. **使用Sqoop导入**：
   ```bash
   sqoop export \
   --connect jdbc:mysql://localhost:3306/dbtaobao \
   --username root \
   --password 123456 \
   --table user_log \
   --export-dir /user/hive/warehouse/dbtaobao.db/inner_user_log \
   --input-fields-terminated-by ','
   ```

### 4. 验证数据

1. **验证MySQL表数据**：`mysql -u root -p123456 -e "select count(*) from dbtaobao.user_log;"`
2. **抽样前10条数据**：`mysql -u root -p123456 -e "select * from dbtaobao.user_log limit 10;"`
3. **检查数据一致性**：确保Hive表和MySQL表的数据行数一致

## 实验结果

- **Hive inner_user_log行数**：100000
- **MySQL user_log行数**：100000
- **数据一致性**：Hive表和MySQL表数据一致
- **数据质量**：抽样前10条无乱码

## 实验结论

实验三Sqoop数据导入/导出完成，成功将Hive中的数据导入到MySQL中，验证了Sqoop在大数据与关系型数据库之间的数据交互能力。

# 实验二：Hive数据分析

## 实验目的

- 熟悉Hive SQL的基本语法
- 掌握Hive数据分析的基本方法
- 了解大数据分析的常见场景

## 实验步骤

### 1. 基础查询

1. **前10品牌**：`hive -e "select brand_id, count(*) as cnt from dbtaobao.user_log where action=3 group by brand_id order by cnt desc limit 10;"`
2. **前20条时间和类别**：`hive -e "select month, day, cat_id from dbtaobao.user_log limit 20;"`
3. **嵌套别名查询**：`hive -e "select brand_id, cnt from (select brand_id, count(*) as cnt from dbtaobao.user_log group by brand_id) t order by cnt desc limit 5;"`

### 2. 统计分析

1. **总行数**：`hive -e "select count(*) from dbtaobao.user_log;"`
2. **去重用户数**：`hive -e "select count(distinct user_id) from dbtaobao.user_log;"`
3. **有效行为数**：`hive -e "select action, count(*) from dbtaobao.user_log group by action;"`
4. **购买用户数**：`hive -e "select count(distinct user_id) from dbtaobao.user_log where action=3;"`

### 3. 高级分析

1. **brand_id=2661 购买量**：`hive -e "select count(*) from dbtaobao.user_log where brand_id='2661' and action=3;"`
2. **点击量前5品牌**：`hive -e "select brand_id, count(*) as cnt from dbtaobao.user_log where action=0 group by brand_id order by cnt desc limit 5;"`
3. **购买转化率**：`hive -e "select (sum(case when action=3 then 1 else 0 end)*1.0/count(*) as conversion_rate from dbtaobao.user_log;"`
4. **男女买家购买数量与占比**：`hive -e "select gender, count(*) as cnt, count(*)*1.0/(select count(*) from dbtaobao.user_log where action=3) as ratio from dbtaobao.user_log where action=3 group by gender;"`
5. **购买次数>5用户**：`hive -e "select user_id, count(*) as buy_count from dbtaobao.user_log where action=3 group by user_id having buy_count>5;"`
6. **各年龄段购买用户分布**：`hive -e "select age_range, count(distinct user_id) as user_count from dbtaobao.user_log where action=3 group by age_range;"`

### 4. 表操作

1. **创建dbtaobao.scan表**：`hive -e "create table dbtaobao.scan as select * from dbtaobao.user_log where action=0;"`
2. **验证表存在**：`hive -e "show tables in dbtaobao;"`
3. **验证表数据**：`hive -e "select count(*) from dbtaobao.scan;"`

## 实验结果

- **总行数**：100000
- **去重用户数**：约80000（具体数值以实际数据为准）
- **购买转化率**：约5%（具体数值以实际数据为准）
- **dbtaobao.scan表**：创建成功，可正常查询
